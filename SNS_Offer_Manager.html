🏦 SNS Offer Manager TEAM TOOL

Add, remove and reorder bank offer images — no coding required

How to use this tool

Follow these 3 steps every time you receive a new bank offer image.

1

Load the website HTML file

Click the area below and select sns_website_final_9.html from your computer.

2

Add / Remove offer filenames

Type the image filename (e.g. hdfc_offer.jpeg) and click Add. Remove any old offers.

3

Download & upload the updated HTML

Click Download — then upload the new HTML file and the image to your server. Done!
